Web3.js ABI loading: Each contract needs its compiled ABI JSON (from artifacts/contracts/ after npx hardhat compile). Import in JS as: const registryABI = require('./artifacts/EquipmentRegistry.json').abi
Contract instances: const registry = new web3.eth.Contract(registryABI, REGISTRY_ADDRESS) — one instance per contract, addresses from .env.
Role detection on connect: On MetaMask connect, call hasRole() for each role constant against the connected address to determine which screens and buttons to enable.
Event subscription (live updates): Use contract.events.EventName({ filter: {assetId} }) for real-time dashboard updates without page refresh.
